## Getting Started

### 1. Clone the Repository

```bash
git clone https://github.com/jatinsce21/Health-Care-Plus.git
```

### 2. Install Dependencies

```bash
npm install --force
```

### 3. Run the Application

#### Backend

```bash

npm start
```

The backend server will run on http://localhost:4000 by default.
